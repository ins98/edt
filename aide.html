E D T

EDT : Emploi du temps universitaire en ligne

   ‟génération automatique,essai gratuit, facile, rapide, entièrement en ligne.”



Pour un bon usage d'EDT, vous devez suivre les étapes de travail :

➤ insérer le nom d'établissement et définir les jours/heures de travail .

➤ insérer les activités et les codes d'activités (cour/tp/td).

➤ insérer les batiments et les salles.

➤ insérer les contraintes.

➤ insérer les enseignants.

➤ insérer les modules.

➤ insérer les promotions (séctions et groupes).

➤ sauvegarder vos données.

➤ consulter les diférents palnning.



Données de base :

Définit le nom d'établissement
(université) vous concevez la
structure de l’horaire en insérant
le nombre de jours de travail
et le nombre maximal de
périodes par jour, ainsi
que d’autres paramètres
de préférence.


Les enseignants :

Saisir nom,prenom d'enseignants
tels que le nombre total maximum
d'enseignants : 6000 .


Les promotions
(séctions et groupes):

Donner les noms de promotion
et le nbr des sections/groupes de
shaque promo et le niveau
(lisence/master) .


Les modules :

lister toutes les matiéres
tels que le nombre est illimité .


Les batiments :

Saisir les Batiments de shaque
département, les salles de shaque
batiments telq que :
Nbr max de bâtiments:6000.
Nbr max de salles:6000.


Les code d'activités :

Donner un code pour
les différent types des modules
ex : C(cour),TP,TD .


Les activités :

un activité est séance de travail
tel que le chef doit choisir
un section / groupe ,un enseignant
un module , un code d'activités
tel que le nbr max d'activités:30000.


Les contraintes :

Saisir les différent contraintes
pour les enseignants,les promotions
les activités,les modules ...
le nbr des contraintes est ilimité .


Consultation :

Vous pouvez consulte les différent
planning et statistiques.
